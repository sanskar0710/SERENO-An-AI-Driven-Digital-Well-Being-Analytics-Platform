# Sereno: AI-Driven Digital Well-Being & Community Platform

A full-stack AI-powered web platform that helps users reflect on their daily thoughts and experiences through text while receiving intelligent insights, mood-improving suggestions, and community support.

## Vision

Sereno is a positive, supportive digital space where users can:
- Write daily reflections
- Analyze emotional patterns with AI
- Receive personalized feel-good suggestions
- Interact with a supportive community
- Share experiences through text, images, and documents
- Track well-being trends over time

The platform aligns with Sustainable Development Goal 3 (Good Health and Well-Being).

## Tech Stack

### Frontend
- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts/Chart.js
- **UI Components**: shadcn/ui

### Backend
- **Framework**: FastAPI (Python)
- **Database**: MongoDB
- **Authentication**: JWT
- **AI/ML**: BERT (Transformers), HuggingFace
- **Real-time**: WebSockets

### Features
- User onboarding & personalization
- Daily reflection/journaling
- AI text analysis engine
- Personalized suggestions
- Community support
- Real-time chat
- Media sharing
- Dashboard & insights

## Project Structure

```
sereno/
├── frontend/          # Next.js frontend
├── backend/           # FastAPI backend
├── ml-models/         # ML models and training
├── shared/            # Shared types and utilities
└── docs/              # Documentation
```

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- MongoDB
- Git

### Installation

1. Clone the repository
2. Set up the backend environment
3. Set up the frontend environment
4. Configure MongoDB
5. Start the development servers
